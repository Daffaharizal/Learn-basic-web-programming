Apa itu HTML?

Link atau Hyperlink adalah elemen HTML yang berfungsi menghubungkan suatu halaman web ke halaman web yang lain.
Elemen ini bisa diklik.. dan nanti akan membuka halaman lain sesui alamat URL yang diberikan.

Cara membuat Link pada HTML

Link pada HTML dapa dibuat dengan tag a, kemudian tag ini WAJIB memiliki atribut Href untuk menentukan alamat URL tujuan dari link.

Berikut ini adalah contoh membuat link pada suatu Paragraf : CONTACT US TENTANG SAYA
